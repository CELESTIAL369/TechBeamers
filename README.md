# EduNote 🚀

EduNote is an AI-powered study assistant that transforms PDFs, handwritten notes, and raw text into smart learning material using AI.

## Features

- AI-generated summaries
- Revision notes
- Quiz generation
- Quick revision cards
- OCR for handwritten notes
- Download handwritten-style PDF notes

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap 5

### Backend
- Node.js
- Express.js

### AI
- Groq API
- Llama 3.1 8B Instant

## Libraries Used

- pdf.js
- Tesseract.js
- html2canvas
- jsPDF
